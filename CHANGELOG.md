# Changelog

## v0.1.4 — 2026-06-26

### 新增

- **桌面级 UI 骨架**：全局 AppShell 顶栏（品牌 + 主题切换 + 设置入口），各页统一 PageHeader 三栏布局
- **UI 基础组件**：Button / Card / Input / Select / Badge / PageHeader 原子组件
- **语义色令牌**：success / warn / danger / neutral 四组 bg+fg 语义色，含深色模式
- **复制按钮**：验证页 A/B 对比框（实时流式 + 静态报告）均加 Copy 按钮
- **Toast 通知**：设置页 Agent 测试结果改用内联 Toast 替代 alert()
- **Favicon**：SVG 图标 + BrowserWindow icon
- **微交互**：按钮 active scale 反馈、hover border-color 过渡

### 优化

- **视觉密度**：全局字号提升（9→11 / 10→12 / 11→13px），滚动条 8px 细窄样式
- **Workbench / PageNav / Modal**：inline style 全部迁移 Tailwind class
- **硬编码颜色**：FlowCanvas / Canvas / Home / Validate / AppShell 中 hex 色值替换为 CSS 变量令牌
- **i18n 完整**：ProposalCard / ConceptTree / Canvas 中硬编码中文替换为 i18n key
- **Modal 统一**：Home 3 个 + Validate 1 个手写弹窗统一改用 Modal.tsx 组件
- **自动更新**：默认关闭自动更新，设置页提供手动检查 + 自动开关

## v0.1.2 — 2026-06-19

### 新增

- **应用内自动更新**：基于 electron-updater，启动时后台静默检查并下载新版，下载完成弹窗提示「立即重启 / 稍后」安装；仅打包安装版生效，发布源为 GitHub Releases
- **深色 / 浅色主题**：设置·常规页新增主题切换，支持「浅色 / 深色 / 跟随系统」三档；跟随系统时实时响应系统明暗变化，偏好持久化、启动即应用

### 修复

- **萃取智能体丢失场景上下文**：萃取开场不再重复追问「你想萃取哪个领域」——引导阶段已确认的场景定义（主体/触发/边界）现会注入萃取与「从文档起草」的提示词，直接切入具体经验

## v0.1.1 — 2026-06-19

### 新增

- **画布节点双击展开编辑**：双击知识积木打开编辑模态，可完整编辑标题与内容；双击流程步骤可单独编辑该步骤
- **连线关系导出 SKILL.md**：导出时把画布连线（关联关系）以「关联：→ 目标块」行内联到来源块下方，关系信息不再在打包时丢失

### 优化

- **界面焕新**：引入 Tailwind CSS（v4），重新设计首页、场景定义、A/B 验证、设置四个页面——统一靛蓝品牌色、聚焦式输入、卡片错位渐入与悬停微交互；经验萃取工作台布局保持不变

### 修复

- 修复 `index.html` 中无 `@layer` 的全局 `*{margin:0}` 重置覆盖 Tailwind 工具类、导致页面间距塌陷的问题

## v0.1.0 — 2026-06-15

首个公开发布版本。

### 核心功能

- **场景管理**：创建、进入、删除场景；支持附件上传（.txt / .md / .pdf / .docx）
- **场景引导**：引导 Agent 通过多步工具调用补全场景上下文（主体/触发/边界），支持单选/多选/自由输入
- **经验萃取**：萃取 Agent 多步工具循环，边对话边操作画布（增量 canvas_add / canvas_update / canvas_delete / propose）
- **七巧板经验画布**（React Flow）：
  - 三类知识积木：流程（蓝）/ 规则（深蓝）/ 洞察（橙）
  - 幽灵积木：智能体提议以半透明虚线呈现，原地采纳/拒绝
  - 元件栏拖入、行内编辑、节点连线、撤销/重做（50步）
  - 小地图 + 缩放
- **缺口驱动追问**：系统追踪三类知识覆盖度，Agent 围绕空白类别设计追问
- **证据分级**：制度级 / 已验证 / 样本级 / 探索性
- **A/B 验证**：裸模型 vs 带 Skill 模型双流并行，五维度差异分析（专业判断/操作性/边界意识/风险提示/幻觉劣化）
- **参考文档**：上传解析、检索注入、「从文档起草」批量建块
- **打包导出**：健康检查 + ZIP（SKILL.md + experience-card.json + 参考文档）
- **设置**：LLM Provider 管理、三智能体独立配置、界面/输出语言切换（中文/English）

### 技术亮点

- AG-UI 事件流双通道（invoke 请求响应 + agent:event 推送）
- Function calling 原生支持 + JSON 降级协议自动切换
- 429 限流专用长退避（5s/21s/30s），超时/5xx 标准退避
- 长对话自动摘要压缩（>24 条消息）
- 增量画布操作（不回吐整卡 JSON，省 token 防丢条目）
- 完全本地，数据不出本机
