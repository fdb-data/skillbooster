# UI/UX 设计

## 设计语言

### 视觉基调

- **主色**：靛蓝 #4F46E5，传达专业、可信、智能
- **背景**：画布白 #FAFAFC，干净留白
- **文字**：正文 #1A1A2E，次要 #6B7280，三级 #A6ABB5
- **分割线**：#ECECF1，轻量分隔
- **圆角**：卡片 10px，胶囊 999px，区块 8px

### 知识类型色板

| 类型 | 颜色 | 语义 | 状态 |
|------|------|------|------|
| 流程 | #3B82F6 蓝 | 怎么做 · 步骤序列 | 启用 |
| 规则 | #2563EB 深蓝 | 怎么判 · 怎么防 | 启用 |
| 洞察 | #E08A2B 橙 | 怎么看 · 信号与模式 | 启用 |
| 概念 | #8B5CF6 紫 | 是什么 · 领域定义 | 暂缓（企业级特性，旧数据仍渲染） |
| 关系 | #10B981 绿 | 怎么连 · 概念关联 | 暂缓（企业级特性，旧数据仍渲染） |

### 智能体回复 Markdown 渲染

引导页与萃取对话的 AI 气泡统一用自研轻量 Markdown 组件（`src/components/Markdown.tsx`，零依赖）渲染：

- 支持：`**加粗**`、`` `行内代码` ``、`---` 分隔线、`#`~`####` 标题、`- ` 无序列表、`|…|` 表格（表头浅灰底，过宽横向滚动）
- 未识别语法按纯文本原样展示，不会渲染坏
- 用户消息保持纯文本（pre-wrap）

### 字号体系

| 用途 | 大小 |

| 级别 | 用途 | 样式 |

## 页面设计

### 一、Home 页（场景首页）

#### 布局

- 顶部：全宽输入区（胶囊输入框 + 发送按钮 + 附件按钮）
- 主体：3 列网格场景卡片
- 底部：社区模板广场入口（灰色占位）

#### 场景卡片

- 卡片结构：名称（大字）+ 证据等级条（彩色横条）+ 状态标签 + 删除按钮
- 交互：hover 显示操作按钮，点击进入场景

#### 附件上传

- 支持点击 📎 或拖拽文件到输入区（.txt/.md/.pdf/.docx）
- 附件以芯片展示：📎 图标 + 文件名（超长省略）+ ✕ 删除；按路径去重
- 开始新项目时附件自动落库为参考文档

---

### 二、Guide 页（场景引导）

#### 布局

- 左侧（60%）：引导对话区
- 右侧（40%）：场景属性面板

#### 对话区

- AI 消息气泡（**流式打字** + Markdown 渲染）+ 选项芯片按钮（2~4 个）
- **选项芯片交互**：
  - 单选：点击直接发送
  - 多选（智能体声明 multiSelect）：芯片勾选切换（✓ 前缀 + 主色高亮）+ "可多选"提示 + 「确认（N）」按钮拼接发送
  - 「其他 · 直接说」：点击展开行内输入框（自动聚焦 + 发送按钮），不直接发送选项文本
- 用户消息气泡
- 思考中状态：三点脉冲 + 实时工具状态文本（"正在更新场景属性…"/"正在准备问题…"）
- **失败反馈**：LLM 调用失败显示友好错误气泡（429 限流提示等待时长），绝不静默
- 步骤指示："场景引导 · step / 3"

#### 属性面板

- 字段：场景名 / 主角 / 触发时机 / 包含 / 排除
- 智能体调工具时**实时填充**（不等本轮结束），用户可随时修改

#### 完成态

- "进入萃取工作台"按钮（primary）

### 三、Workbench 页（萃取工作台）

#### 布局

- 顶栏：场景名 + 阶段导航 **场景定义（ghost）| 经验萃取（soft 高亮，当前页）| 验证（ghost）| 导出/部署（primary）**；点「场景定义」返回引导页（场景草稿自动恢复）
- 左栏（216px）：参考文档面板，**可收起**——「«」收起为 28px 竖条（» + 竖排"参考文档" + 文档数），点击展开
- 中栏（自适应）：七巧板经验画布
- 右栏（340px）：萃取对话

#### 七巧板经验画布（React Flow）

- **知识积木**（240px 宽节点）：
  - 左侧 4px 分类色条 + 类型标签胶囊 + 证据等级圆点 + 标题（单行省略）
  - 内容最多 3 行截断
  - 底部：待验证/刚长出标签 + ✎ 编辑 / 🗑 删除按钮
  - 上下各一个连接点（按类型着色）
- **编辑态**：节点内 input + textarea + 证据等级四挡胶囊 + 保存/取消
- **幽灵积木**（提议）：
  - 半透明白底（0.82）+ 1.5px 类型色虚线边框
  - "类型·提议"标签 + 灰字内容
  - 操作：✓ 采纳 / ✎ 改 / ✕ 不对（红字）
  - 入场动画 `ghostIn`（缩放 0.92→1 + 上浮淡入 0.35s）
- **元件栏**（Panel 左上）：白底卡片，三个可拖拽部件——流程/规则/洞察（色条 + 名称 + 一句话提示），cursor: grab；概念/关系暂不开放
- **进度条**（Panel 右上）：胶囊容器内 5 圆点进度 + 块数 + ↩ 撤销 / ↪ 重做
- **连线**：流程顺序=蓝色动画线，关联=灰色实线；选中按 Del 删除
- **小地图**（左下，120×80，节点按类型着色）+ 缩放控制（右下）+ 16px 网格背景
- 新块动画：`blockGrowIn`（上浮 8px + 淡入 0.3s）+ 主色边框高亮，3 秒淡出

#### 萃取对话

- **开场引导卡**（对话为空时）：自我介绍 + 三种用法（直接讲经验 / 上传文档起草 / 元件栏手搭）+「不知道从哪说起？让智能体先提问」primary 按钮
- 消息列表 + 流式气泡 + 底部输入框
- **流式渲染**：智能体回复逐字上屏（独立气泡，Markdown 渲染含表格）
- **工作状态行**：三点脉冲 + 状态文本（"正在添加条目…"/"思考中…"）+ ⏹ 中断按钮（胶囊）
- **提议提示**：主色浅底横条"🧩 N 条提议已作为虚线积木放上画布，请在画布上采纳或拒绝"

### 四、Validate 页（A/B 验证）

#### 布局

- 顶栏：场景名 + 返回按钮
- 主体：左右对比面板

- **维度卡片网格**（auto-fill, minmax 220px）：蓝底卡片 + 左侧蓝色条，维度名 + 分析文本
- **总评条**：主色浅底加粗
- 旧格式降级：整段 diffSummary 蓝底展示
- "保存为验证记录"按钮（saved 后变 "✓ 已保存"）

## 交互规范

### 流式与状态反馈

- 智能体响应全程流式：文本增量逐字上屏
- 工具调用映射为人话状态：canvas_add→"正在添加条目…"、propose→"正在生成提议…"、update_scene→"正在更新场景属性…"
- 长操作随时可中断（⏹ 按钮），中断不报错（静默清理）

### 加载状态

- 按钮点击后禁用 + 文字变为"处理中..."
- 验证页骨架屏仅在首字到达前显示

### 错误处理

- 全局错误条：顶部红色横条，5 秒自动消失
- IPC 错误通过 IpcResult 信封统一处理
- 用户主动中断（ABORTED）不显示错误
- 引导对话失败：对话区内显示友好错误气泡（不依赖全局错误条）；429 限流给出"等半分钟再发"的具体指引

### 确认操作

- 删除场景：二次确认弹窗
- 其他删除：直接执行（画布操作可撤销兜底）

### 快捷键

- Ctrl+Enter：发送消息（Home 页输入框）
- Del / Backspace：删除选中连线（画布）

### 滚动行为

- 对话列表自动滚动到底部（含流式过程中）
- 新消息进入时平滑滚动

### 动画反馈

| 动画 | 触发 | 效果 |
|------|------|------|
| blockGrowIn | 新增画布条目 | 上浮 8px + 淡入 0.3s，主色高亮 3 秒淡出 |
| ghostIn | 幽灵积木出现 | 缩放 0.92→1 + 上浮淡入 0.35s |
| pulse | 思考中/骨架屏 | 透明度脉冲 |
